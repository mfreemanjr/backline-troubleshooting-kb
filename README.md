# PowerScale OneFS Operations Library (MkDocs)

This repository hosts the **PowerScale / OneFS** operational KB. It is built with **MkDocs** and **Material for MkDocs**.

## Local preview
```bash
pip install mkdocs-material
mkdocs serve
```
Open http://127.0.0.1:8000

## Deploy (GitHub Pages)
- Ensure default branch is **main**
- Enable Actions (organization policy may apply)
- Push to **main** — the `deploy` workflow builds and publishes to `gh-pages`
